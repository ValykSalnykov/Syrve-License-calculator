# Калькулятор лицензий Syrve

## Описание проекта

Калькулятор лицензий Syrve — это интерактивное веб-приложение, предназначенное для автоматизации процесса подбора и расчета стоимости лицензий системы управления ресторанным бизнесом Syrve. Приложение помогает менеджерам по продажам и потенциальным клиентам определить оптимальную конфигурацию лицензий на основе специфических требований бизнеса.

### Целевая аудитория
- Менеджеры по продажам Syrve
- Бизнес-консультанты
- Владельцы и управляющие ресторанов, кафе, баров
- Руководители сетей общественного питания

### Основная цель
Упрощение процесса подбора лицензий и формирование точного коммерческого предложения с учетом всех необходимых модулей, количества точек продаж и функциональных требований бизнеса.

## Функциональные возможности

### 1. Панель настроек

#### 1.1 Выбор типа лицензии
Калькулятор поддерживает следующие уровни лицензий Syrve:
- **Basic (Базовый)** — базовая функциональность POS-системы
- **Professional (Профессиональный)** — расширенные возможности управления
- **Enterprise (Корпоративный)** — полный функционал для сетевых заведений

**Зависимости:**
- Выбор типа лицензии определяет доступность дополнительных модулей
- От типа лицензии зависит базовая стоимость на одну точку продаж
- Некоторые модули доступны только в определенных типах лицензий

#### 1.2 Настройки валюты и периода
- Выбор валюты расчета (RUB, USD, EUR, GBP)
- Выбор периода подписки (месяц, год)
- Автоматический пересчет при изменении настроек
- Отображение скидки при годовой оплате

#### 1.3 Дополнительные параметры
- Включение/выключение НДС в расчет
- Применение корпоративных скидок
- Настройка отображения (свернутый/развернутый вид)

### 2. Выбор фронтов (точек продаж)

#### 2.1 Добавление точек продаж
Пользователь может добавить несколько точек продаж (фронтов), для каждой из которых указываются:
- **Название точки** (например: "Главный зал", "Бар", "Терраса")
- **Тип точки** (Ресторан, Кафе, Бар, Food Truck, Доставка)
- **Количество терминалов** на точке
- **Количество сотрудников** с доступом к системе

#### 2.2 Управление точками
- Добавление неограниченного количества точек
- Редактирование параметров существующих точек
- Удаление точек
- Клонирование конфигурации точки
- Быстрое переключение между точками

**Зависимости:**
- Стоимость лицензии умножается на количество терминалов
- Некоторые модули рассчитываются на точку, другие — на организацию
- Количество сотрудников влияет на стоимость модуля управления персоналом

### 3. Выбор модулей и функций

#### 3.1 Основные модули

**Модуль управления запасами**
- Автоматический учет остатков
- Контроль закупок и списаний
- Инвентаризация
- *Зависимость: доступен с Professional и выше*

**Модуль онлайн-заказов**
- Мобильное приложение для клиентов
- Интеграция с сайтом
- Онлайн-оплата
- *Зависимость: требует модуль CRM*

**Модуль доставки**
- Управление курьерами
- Интеграция с агрегаторами (Яндекс.Еда, Delivery Club)
- Отслеживание заказов
- *Зависимость: требует модуль онлайн-заказов*

**Модуль бронирования столов**
- Онлайн-бронирование
- Управление залом
- Схема зала
- *Зависимость: доступен с Professional и выше*

**Модуль аналитики и отчетности**
- Расширенные отчеты
- Дашборды в реальном времени
- Прогнозирование продаж
- *Зависимость: доступен только в Enterprise*

**Модуль управления персоналом**
- Учет рабочего времени
- Расчет зарплат и премий
- Оценка эффективности
- *Стоимость зависит от количества сотрудников*

**Модуль лояльности**
- Программы лояльности
- Бонусные карты
- Промо-акции
- *Зависимость: требует модуль CRM*

#### 3.2 Интеграции

**Интеграция с 1С**
- Синхронизация данных
- Выгрузка отчетности
- *Зависимость: доступна с Professional*

**Интеграция с агрегаторами доставки**
- Uber Eats, Deliveroo, Just Eat
- *Зависимость: требует модуль доставки*

**Платежные системы**
- Эквайринг
- Онлайн-платежи
- Безналичные расчеты

**Интеграция с кассовым оборудованием**
- Фискальные регистраторы
- Терминалы оплаты
- Весовое оборудование

### 4. Логика зависимостей модулей

Калькулятор автоматически управляет зависимостями между модулями:

```
Схема зависимостей:

Базовая лицензия (Basic)
├── Доступны базовые функции POS
└── Ограниченный набор интеграций

Professional
├── Все функции Basic
├── + Управление запасами
├── + Бронирование столов
├── + CRM
│   ├── → позволяет Модуль лояльности
│   └── → позволяет Онлайн-заказы
│       └── → позволяет Модуль доставки
└── + Интеграция с 1С

Enterprise
├── Все функции Professional
├── + Расширенная аналитика
├── + Управление сетью заведений
├── + Централизованное управление меню и ценами
└── + Приоритетная поддержка 24/7
```

#### Правила автоматической активации:
- При выборе "Модуль доставки" автоматически добавляется "Модуль онлайн-заказов" и "CRM"
- При выборе "Модуль лояльности" автоматически добавляется "CRM"
- При выборе интеграции с агрегаторами автоматически добавляется "Модуль доставки"

#### Правила блокировки:
- Если выбрана лицензия Basic, модули Professional/Enterprise недоступны (отображаются серым)
- При снятии базового модуля автоматически снимаются все зависимые модули
- Появление предупреждающих сообщений о необходимых зависимостях

### 5. Расчет стоимости

#### 5.1 Алгоритм расчета
```
Итоговая стоимость = 
    (Базовая стоимость лицензии × Количество терминалов) +
    Σ(Стоимость модуля × Множитель*) +
    Стоимость интеграций +
    Стоимость обучения (опционально) +
    Стоимость внедрения (опционально)
    
* Множитель зависит от типа модуля:
  - "на терминал" — умножается на количество терминалов
  - "на точку" — умножается на количество точек
  - "на организацию" — фиксированная цена
  - "на пользователя" — умножается на количество сотрудников
```

#### 5.2 Отображение расчета
- **Промежуточные итоги** по каждой категории
- **Детализация** по точкам продаж
- **Визуализация** структуры стоимости (диаграммы)
- **Сравнение** вариантов конфигураций

### 6. Вывод результатов

#### 6.1 Итоговая сводка
Отображает:
- Общую стоимость в месяц/год
- Стоимость по каждой точке продаж
- Детализацию по модулям
- Список всех выбранных опций
- Условия и ограничения

#### 6.2 Экспорт данных
Поддерживаемые форматы:
- **PDF** — готовое коммерческое предложение с логотипом
- **Excel** — детальная смета с формулами
- **JSON** — для сохранения конфигурации
- **Ссылка** — для совместного использования расчета

#### 6.3 Шаблоны документов
- Коммерческое предложение
- Техническое задание
- Договор (шаблон)
- Сравнительная таблица конфигураций

## Бизнес-логика и правила

### Ценообразование

#### Базовые цены лицензий (EUR/месяц на терминал):
- Basic: 49 EUR
- Professional: 79 EUR
- Enterprise: 99 EUR

#### Скидки:
- Годовая подписка: -15%
- 5-10 терминалов: -5%
- 11-20 терминалов: -10%
- 21+ терминалов: -15%
- Корпоративный клиент: -20% (по запросу)

#### Дополнительные услуги:
- Внедрение (единоразово): от 500 EUR
- Обучение персонала (на человека): 100 EUR
- Миграция данных: от 300 EUR
- Приоритетная поддержка: +20% к стоимости лицензии

### Ограничения и условия

**Минимальные требования:**
- Basic: минимум 1 терминал
- Professional: рекомендуется от 2 терминалов
- Enterprise: минимум 3 точки продаж

**Ограничения по модулям:**
- Управление сетью: только Enterprise
- Расширенная аналитика: только Enterprise
- Базовая аналитика: Professional и выше
